# Corner Grocer Item Tracking System

## Project Overview

The Corner Grocer Item Tracking System is a C++ application developed to help a grocery store analyze daily purchasing trends. The program reads a text file containing grocery items purchased throughout the day, counts the frequency of each item, and presents the information in several easy-to-read formats. This allows the business to make informed decisions about inventory management and product placement.

---

## Problem Statement

The Corner Grocer needed a way to automate the process of counting how often each grocery item was purchased. Instead of manually reviewing hundreds of transactions, this application processes the sales data instantly and produces reports that can assist management in organizing inventory more efficiently.

---

## Features

- Reads grocery purchase data from a text file
- Counts the frequency of every purchased item
- Searches for the frequency of a specific item
- Displays all item frequencies
- Generates a histogram using asterisks (*)
- Creates a backup data file (frequency.dat)
- Interactive menu-driven interface
- Input validation for user selections

## Challenges

One of the biggest challenges during development was correctly reading the input file while simultaneously updating the frequency count of each grocery item. Learning how to use the C++ map container efficiently required research and experimentation. Breaking the project into smaller functions made debugging much easier.
